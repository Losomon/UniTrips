# Contributing to UniTrips

Thank you for your interest in contributing to UniTrips! 🎉

## How to Contribute

1. **Fork the repository**
2. **Create your feature branch** (`git checkout -b feature/amazing-feature`)
3. **Commit your changes** (`git commit -m 'Add some AmazingFeature'`)
4. **Push to the branch** (`git push origin feature/amazing-feature`)
5. **Open a Pull Request**

## Development Setup

### Frontend
```
cd frontend
# Just open index.html in browser
# Edit HTML/CSS/JS directly
```

### Backend (JSP)
```
# Copy backend/ to Tomcat webapps/
# Access: http://localhost:8080/backend/tripForm.jsp
```

## Pull Request Guidelines

- Fill out the **template** below
- Keep PRs focused on **one change**
- Update relevant **documentation**
- Add **tests** if applicable
- Follow existing **code style**

## PR Template

```
**What:**
- Brief description

**Why:**
- Motivation/problem solved

**How to test:**
1. Steps to verify
2. Expected result

**Changes:**
- File changes summary

**Checklist:**
- [ ] Tested locally
- [ ] Docs updated
- [ ] No breaking changes
```

## Code Style

```js
// JS (frontend)
const camelCase = true;
const maxLineLength = 100;
const singleQuotes = true;
```

```jsp
<%-- JSP (backend) --%>
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%-- 2-space indent, JSTL preferred --%>
```

## Issues

- 🐛 **Bug** - Unexpected behavior
- ✨ **Feature** - New functionality  
- 🔧 **Improvement** - Refactor/optimization
- 📖 **Docs** - Documentation only
- ❓ **Question** - Usage/support

## Questions?

Open an issue on GitHub for questions or support.

Happy contributing! 🙌
